
## Đối tượng Phòng

### Thuộc tính
- soPhong: String
- trangThai: boolean
- tienDienNuoc: double

### Chức năng CRUD
- Create: thêm phòng mới (mặc định trạng thái trống)
- Read: tìm theo số phòng hoặc danh sách phòng trống
- Update: sửa thông tin (tiền điện nước, trạng thái)
- Delete: xóa phòng theo số phòng
